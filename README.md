# Backend

## Project Setup

### Python 3 

Make sure you have Python 3 installed

### Django

Install Django using

```
pip3 install django
```
### Run the project

You have to be in the /emotionalstoryteller/backend folder, and you have to run 

```
python3 manage.py runserver
```
